Código-Fonte / Pré Formatação

Indentação: é o espaçamento aplicado no início das linhas de código que ajuda a manter uma hierarquia visual.

Formatação: No HTML é um recurso que deixa seu texto padronizado e de forma a deixar a leitura mais agradável.

O comando get.ElementsById('teste') é escrito em JavaScript. ⇐ Texto sem Formatação

O comando get.ElementsById('teste') é escrito em JavaScript. ⇐ Texto com Formatação

        





1) Exemplo de Código com identação e com formatação: maneira correta


    
var n = prompt("Digite um numero"); 
var total = n/2; if(n / 2 = 0) { alert("Par"); } 
if(n / 2 = 1){ alert("Impar"); } 
alert(total);
        
    



    
2) Exemplo de Código sem Identação e sem Formatação:

   
    var n = prompt("Digite um numero"); var total = n/2; if(n / 2 = 0) { alert("Par"); } if(n / 2 = 1){ alert("Impar"); } alert(total);


3) Exemplo de Código sem  Identação e com Formatação:


    var n = prompt("Digite um numero"); var total = n/2; if(n / 2 = 0) { alert("Par"); } if(n / 2 = 1){ alert("Impar"); } alert(total);
    
    
4) Exemplo de Código com  Identação e sem Formatação:


    
        var n = prompt("Digite um numero");
        var total = n/2; if(n / 2 = 0) { alert("Par"); }
        if(n / 2 = 1){ alert("Impar"); }
        alert(total);
