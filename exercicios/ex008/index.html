Principais Formatações do HTML:


Negrito / Destaque

Nesta frase temos um Termo em negrito usando a tag <b> (nao semantico), apenas forma.

Nesta frase, tenho um Termo em Destaque usando a tag <STRONG> (semântica).

Itálico / Ênfase

Nesta frase, temos um Termo em Itálico usando a tag <I> (não semântica).

Nesta frase, temos um Termo em Itálico usando a tag <EM> (semântica).

Texto Marcado

Podemos criar um texto marcado usando <MARK>

Podemos também criar uma marcação com todas as marcações da mesma cor, utilizando o CSS no <head>. Pois com isso, irá selecionar todos da mesma cor. Como no código: 

            
                <head>
                <style>
                    mark {
                        background-color: blue;
                    }
                </style>
                </head>

Podemos criar também um texto marcado de forma individual usando <mark style="background-color: + a cor em código hexadecimal

Texto Deletado

Podemos marcar um texto e como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Com a tag <ins> (semântico)

Existe também o sublinhado com a tag <U> (não semântico)

Texto sobrescrito

Para inserir coisas como: x20 + 3. Com a tag <sup>

Texto Subscrito

Para inserir coisas do tipo: H2O. Com a tag <sub>
